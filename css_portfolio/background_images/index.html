CSS background-image

The following picture is not able to fit, but there is a way to make it fit using html

Now you can see the picture! Using background-size: cover;

Here the picture of the same smiley face, but reduce the size of the window and there should be another face underneath.This was done by using background-image: contain;.




Here the Smiley Face is repeating using the background-repeat:repeat method.




Here the same Smiley Face is not repeating and it is done by using background-repeat:no-repeat;




Here the Smiley Face is put into the center of the square using background-position:center;

But there are other ways to put the image within the square, such as to the right or the left.
